# Implementation Report: T-003

Report Schema: implementation-report/v2

## Target

Implement the immutable, canonical, hash-bound per-run task handoff contract.

Added a closed task-input schema plus paired Bash/PowerShell snapshot builders
and validators. Review remediation made published snapshots read-only, rejects
symlinks at every path component, validates real UTC calendar timestamps,
rejects all input/output ancestry overlaps, and publishes with atomic
