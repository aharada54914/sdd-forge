# Implementation Report: T-002

Report Schema: implementation-report/v2

## Target

Implement deterministic turn-first routing, one-tier recurrence escalation, and
hash-bound human resume after terminal-tier blocking.

## Summary

Added paired Bash/PowerShell model selectors and terminal-resume validators.
The selectors minimize expected attempts before choosing the weakest sufficient
provider-neutral tier and then cost. Repeated equal classified failures advance
exactly one tier; recurrence at the strong tier remains blocked. Resume requires
a trusted persisted blocked state, a revised task-section hash, diagnosis
evidence, and matching human reapproval. Three independent review rounds found
and resolved state-binding, runtime-parity, audit-output, and symlink-containment
defects.
