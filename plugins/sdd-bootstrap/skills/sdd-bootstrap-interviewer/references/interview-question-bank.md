# Interview Question Bank

## Product Questions

- 何を作りたいですか？
- それは誰のどんな課題を解決しますか？
- 既存の代替手段は何ですか？
- そのソフトが成功したと言える状態は何ですか？
- 最初のMVPで絶対必要な機能は何ですか？
- MVPに入れない機能は何ですか？

## User and Role Questions

- 利用者は誰ですか？
- 管理者はいますか？
- 承認者、閲覧者、外部ユーザーはいますか？
- 権限ごとにできることは何ですか？

## Data Questions

- 主なデータは何ですか？
- 個人情報は扱いますか？
- 添付ファイルは扱いますか？
- データ保持期間はありますか？
- CSV/Excelインポート・エクスポートは必要ですか？

## Workflow Questions

- ユーザーは最初に何をしますか？
- 通常フローは何ですか？
- 異常系は何ですか？
- 承認、差戻し、キャンセルはありますか？

## Architecture Questions

- Webアプリですか？APIですか？CLIですか？
- フロントエンドは必要ですか？
- バックエンドは必要ですか？
- DBは必要ですか？
- 認証はどうしますか？
- 外部システム連携はありますか？
- デプロイ先はどこですか？
