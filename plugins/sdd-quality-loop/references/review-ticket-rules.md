# Review Ticket Rules

Review feedback must be converted into structured review tickets.

## Ticket Fields

- ticket_id
- type
- severity
- target
- summary
- problem
- expected_fix
- references
- auto_fix_allowed
- requires_human_decision

## Types

- bug
- test-gap
- security
- spec-drift
- ci-failure
- api-contract-mismatch
- documentation
- refactor-request

## Rule

Fix only what the ticket explicitly asks for.
